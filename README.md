# MAHAL 💙

A mobile-first private birthday celebration web application for Jane.

## Features
- 🎂 Personalized countdown timer to her birthday
- 💌 Interactive sliding love letter with a typewriter animation
- 💖 Dynamic deck of swipeable "Reasons Why I Love You" cards
- 📸 Horizontal memories gallery carousel synced with a vertical journey timeline
- 🎁 Interactive surprise gift box revealing a custom promise card and couple photo
- 🎵 Sweet, bell-like synthesized background music box chords
- ⚙️ Full settings panel to edit/add/delete memories, change the countdown date, edit the letter text, and more
- 💾 Dual-key Local Storage persistence to ensure custom memories are saved permanently

## Tech Stack
- HTML5, CSS3, Pure JavaScript
- Web Audio API (for offline music synthesis)
- HTML5 Canvas (for compressed photo storage and confetti physics)
